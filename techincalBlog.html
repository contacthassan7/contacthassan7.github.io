What is the difference between Margin, border and padding?

Margin is the starting point of the element, it changes the placement of the element on the page. Margins are used to add spaces between an image and the description of that image.

Border is outline of the element, could be as big as a page itself or as little as a button

Padding is the distance between the border of the element and its content, below is the example of padding CSS Padding is used if we want to create a space between an element and the edge of the container or the border.
